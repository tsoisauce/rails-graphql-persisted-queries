# Rails GraphQL Persisted Queries

Ruby on Rails project to demo HTTP caching strategies.

## Installation

Bundle install

```bash
bundle
```

Webpack install

```bash
rails webpacker:install
```

Yarn install

```bash
yarn
```

GraphQL install (if needed)

```bash
rails generate graphql:install
```

## Run Server

To start Rails server:

```bash
rails s
```
